Release Summary 2.1.0⚓︎

Each release of the HID Origo SDK includes stability and performance enhancements. The following sections provide details on features included in this release.

Release Details⚓︎

The HID Origo SDK is integrated with the HID Origo Sample App.

HID Origo SDK 2.1.0 for Android contains the following new features:

  • Supports HID Identity Positioning Reader Approaches (Continuous read, Read on demand). (Please refer to HID Identity Positioning for more details).
